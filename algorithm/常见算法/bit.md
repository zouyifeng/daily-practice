### 位运算

#### 按位与
每一位都为 1，结果才为 1

```js
8 & 7 // -> 0
// 1000 & 0111 -> 0000 -> 0
```

#### 按位或
其中一位为 1，结果就是 1
```js
8 | 7 // -> 15
// 1000 | 0111 -> 1111 -> 15
```

#### 按位异或
每一位都不同，结果才为 1
```js
8 ^ 7 // -> 15
8 ^ 8 // -> 0
// 1000 ^ 0111 -> 1111 -> 15
// 1000 ^ 1000 -> 0000 -> 0
```

#### 左移 <<
```js
10 << 1 // -> 20
```

#### 右移 >>
```js
10 >> 1 // -> 5 
```
